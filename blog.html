Discuss the scope of var, let, and const

Var:
Function Scope: Variables declared with var have function-level scope. This means they are only accessible within the function in which they are declared, regardless of where within the function they are defined.

Let:
Block Scope: Variables declared with let have block-level scope, meaning they are limited to the block (enclosed by curly braces) in which they are defined. This includes if statements, loops, and functions

Const:
Block Scope: cons have also block Scope.

What do you mean by REST API?

REST API is a collection of guidelines and practices for developing and utilizing HTTP to communicate with web services. It carries out operations including data retrieval, creation, update, and deletion using resource-based URLs, stateless communication, and normal HTTP methods. Data communication between clients and servers using scalable and standardized RESTful APIs is common.
